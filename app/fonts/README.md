# TT Hoves Font Files

Please add the TT Hoves font files to this directory:

- `TTHoves-Regular.woff2`
- `TTHoves-Medium.woff2`
- `TTHoves-Bold.woff2`

You can purchase TT Hoves from TypeType foundry or use your licensed version.

If you don't have the font files yet, the site will fall back to system sans-serif fonts.
